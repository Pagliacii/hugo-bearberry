## Python

```python
def hello_world():
    print("Hello, world!")
```

## C

```c
#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
```

## C++

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}
```

## Go

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, world!")
}
```

## JavaScript

```javascript
console.log("Hello, world!");
```

## Rust

```rust
fn main() {
    println!("Hello, world!");
}
```
