Math Typesetting via Mathjax

Published: 2024-01-14 00:00:00 | Updated: 2024-01-14 15:48:34 | 100 words | 1 mins

Mathematical notation in a Hugo project can be enabled by using third party JavaScript libraries.

Configuration

You can enable Mathjax by using the mathjax option in your site’s configuration file or in the front matter:

1math = true
2mathjax = true

Example

When \(a \ne 0\), there are two solutions to \(ax^2 + bx + c = 0\) and they are $$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$
